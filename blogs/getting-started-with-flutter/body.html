Flutter is a popular open-source UI toolkit for building high-performance, cross-platform mobile and web apps. With Flutter, you can write code once and deploy it to both Android and iOS devices, as well as the web, macOS, Windows, and Linux. In this guide, we'll walk you through the steps to get started with Flutter.

Step 1: Install Flutter

To get started with Flutter, you'll need to install it on your computer. Here's how:

  1. Download the Flutter SDK from the official Flutter Website.
  2. Extract the contents of the downloaded ZIP file to a location on your computer.
  3. Add the Flutter SDK to your system path. To do this, open your terminal and type:
export PATH="$PATH:[PATH_TO_FLUTTER_DIRECTORY]/bin"

4. Verify that Flutter is installed correctly by running the following command:

flutter doctor

This command will check your system and show you any issues that need to be resolved before you can start building apps with Flutter.

Step 2: Set up an IDE

Flutter can be used with a variety of IDEs, including Android Studio, Visual Studio Code, and IntelliJ IDEA. In this guide, we'll use Visual Studio Code (coz that's what I use).

  1. Download and install Visual Studio Code from the official website.
  2. Install the Flutter and Dart plugins for Visual Studio Code. To do this, open Visual Studio Code and go to the Extensions panel (Ctrl+Shift+X). Search for "Flutter" and "Dart" and install the plugins.
  3. Once the plugins are installed, you can create a new Flutter project by going to the command palette (Ctrl+Shift+P) and typing "Flutter: New Project". Follow the prompts to create a new project.

Step 3: Build your first Flutter app

Now that you have everything set up, it's time to build your first Flutter app. Here's how:

  1. Open the main.dart file in your project's lib folder.
  2. Replace the default code with the following:
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'My First Flutter App',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Welcome to Flutter'),
        ),
        body: Center(
          child: Text('Hello World'),
        ),
      ),
    );
  }
}

This code creates a basic Flutter app that displays a "Hello World" message on the screen.

  1. Save the file and run the app by typing the following command in your terminal:
flutter run

This will launch the app in an emulator or on a connected device.

Congratulations! You've just built your first Flutter app.
